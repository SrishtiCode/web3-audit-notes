# resources

Useful tools, databases, and references for Web3 security research.

## report databases

| site | what it has |
|---|---|
| [Solodit](https://solodit.xyz) | aggregated findings from all major audit firms |
| [Code4rena](https://code4rena.com/reports) | public contest reports |
| [Sherlock](https://audits.sherlock.xyz) | contest reports + judging rationale |
| [Immunefi](https://immunefi.com/explore/) | bug bounty programs |
| [DeFiHackLabs](https://github.com/SunWeb3Sec/DeFiHackLabs) | PoC exploits for real hacks |
| [Rekt News](https://rekt.news) | post-mortems written for public |
